Sensors and Microcontrollers-Basic

This course offers basic level project based learning of sensors and microcontrollers. In this course children will learn to communicate with the real world through sensors, they will learn the concept of microcontroller and it's usage in the real world. The complete course gives children an insight into the working of electronic circuits, building circuits on bread board, debuggng the circuit, basic programming concepts, writing simple code and interacting with the hardware using in-built software. Worksheets with assignments will be given in each session for children to recap the concepts learnt in the class. At the end of the course children will be assigned projects to test their understanding.

Sensors and Microcontrollers-Advance

This course offers advance level project based learning of sensors and microcontrollers. Children will learn advance electronic components, their usage and theor applications in the real world through projects. The complete course gives children an insight into the working of microcontollers, it's programming and controlling advance circuits. Worksheets with assignments will be given in each session for children to recap the concepts learnt in the class. At the end of the course automated projects using these concepts.

Basics of Electronics and Programming

This course offers advance level project based learning of basic electronics and basics of programming concepts using SCRATCH. This program is designed for beginners and focuses on basic electronic circuits, sensors. Here we also introduce the basic concepts of programming. SCRATCH is used to execute the programs. Throughout the course children and given hands-on electronic projects to understand circuits.Projects using SCRATCH help the children to get familiarized with programming.
